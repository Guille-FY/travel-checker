Travel Tracker is an interactive web application designed for travelers to keep a digital record of their journeys. With a beautiful and responsive map interface, you can easily mark the countries you have visited, track your progress, and visualize your global footprint.

## How to Run Locally

Follow these simple steps to get the application running on your local machine:

1. **Install Dependencies**
    Open your terminal in the project directory and run:

    ```bash
    npm install
    ```

2. **Start the Development Server**
    Once installed, start the application with:

    ```bash
    npm run dev
    ```

3. **Open in Browser**
    Visit [http://localhost:3000](http://localhost:3000) to see your Travel Tracker in action.
